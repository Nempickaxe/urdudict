## Urdudict

Lookup word meanings in Urdu from your terminal


### Example

```
$ urdudict ulfat
Found meaning
~~~~~~~~~~~~~
ulfat (الفت) - love, friendship, affection 

Did you mean
~~~~~~~~~~~~
lift ulfate.n ulfato.n

Source: rekhta.org
```
